# 🛡️ Insurance Analytics Capstone Project | SQL + Power BI + DAX

## 📌 Project Overview

This **Capstone Project** demonstrates an end-to-end Insurance Analytics solution built using **MySQL, Power BI, DAX, and Power Query**. The objective was to transform raw insurance data into actionable business insights through interactive dashboards and KPI reporting.

The project analyzes **45,000+ insurance records** across policy performance, claims management, portfolio health, customer risk profiling, and fraud detection to support data-driven decision-making.

---

## 🎯 Project Objectives

- Analyze overall insurance portfolio performance.
- Track policy lifecycle and customer retention.
- Monitor claims processing efficiency.
- Identify high-risk customers using health indicators.
- Detect fraud trends across policies and regions.
- Build interactive dashboards for business stakeholders.
